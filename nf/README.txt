This directory contains tables of totally real number fields of small degree and root discriminant.
These tables have been generated by John Voight, see

John Voight, Enumeration of totally real number fields of bounded root discriminant,
Algorithmic number theory (ANTS VIII, Banff, 2008), eds. Alfred van der Poorten and Andreas Stein,
Lecture Notes in Comp. Sci., vol. 5011, Springer, Berlin, 2008, 268-281.

The tables can be obtained from 
https://math.dartmouth.edu/~jvoight/nf-tables/index.html .
